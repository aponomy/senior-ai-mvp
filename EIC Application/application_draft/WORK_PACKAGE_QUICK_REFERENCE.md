## Pilot Program Overview

### Target
- **Sites:** 10-15 operational environments
- **Users:** 250-450 elderly (age 65-85+)
- **Duration:** 8-12 weeks per site
- **Model:** Paid pilots (€300-600 per participant)

### Mix
- Swedish municipalities: 6-8 pilots
- Norwegian municipalities: 2-3 pilots
- Private care providers: 4-5 pilots

### Timeline
- M5-M10: Recruitment (20+ LOIs)
- M7-M9: First wave (5 sites)
- M10-M12: Second wave (5 sites)
- M12-M15: Third wave (3-5 sites if needed)

### Evaluation
- Task completion rates (target: >80%)
- Time savings (target: 50-80%)
- Weekly active rate (target: >60%)
- SUS (target: >70)
- NPS (target: >30)
- Safety record (zero sentinel events)
