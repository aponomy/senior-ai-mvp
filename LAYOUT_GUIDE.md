## Object Positioning Rules

### Alignment Priority (Order of Processing)

1. **TOP** â†’ Search Field
   - 100% width
   - Fixed 80px height
   - No horizontal space sharing
   - Can have objects below

2. **BOTTOM** â†’ Timeline  
   - 100% width
   - Fixed 180px height
   - No horizontal space sharing
   - Can have objects above

3. **RIGHT** â†’ Chat Window
   - Max 300px width
   - 100% height (excluding top/bottom objects)
   - No vertical space sharing
   - Takes full height

4. **LEFT** â†’ Large Buttons, Cluster Card, Topics
   - Fill remaining width (after right objects)
   - Fill remaining height (after top/bottom objects)
   - Can share space with right-aligned objects

### Space Calculation Example

```
Viewport: 1920 Ã— 1080
Footer: 80px (always visible)

With Timeline + Chat active:

1. Calculate working height:
   1080 - 80 (footer) = 1000px

2. Process BOTTOM (Timeline):
   Timeline takes 180px from bottom
   Remaining: 1000 - 180 = 820px height

3. Process RIGHT (Chat):
   Chat takes 300px from right, 820px height
   Remaining: 1620px width Ã— 820px height

4. Process LEFT (other objects):
   Objects get: 1620px width Ã— 820px height
```

### Z-Index Layers

```
Layer 1000: Footer (always on top)
Layer 100+: Top-aligned objects
Layer 100+: Bottom-aligned objects  
Layer 90+:  Right-aligned objects
Layer 80+:  Left-aligned objects
Layer 0:    Background
```

## Transition States

### Appearing

```
Before:
  opacity: 0
  transform: scale(0.95)

After:
  opacity: 1
  transform: scale(1)
  
Duration: 300ms cubic-bezier(0.4, 0, 0.2, 1)
```

### Disappearing

```
Before:
  opacity: 1
  transform: scale(1)

After:
  opacity: 0
  transform: scale(0.95)
  
Duration: 300ms cubic-bezier(0.4, 0, 0.2, 1)
```

### Position Changes

```
All position/size changes animate smoothly:
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1)
```
